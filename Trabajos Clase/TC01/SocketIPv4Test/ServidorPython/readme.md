# Pruebas de Conectividad con Servidor Socket

Este readme describe los pasos para realizar pruebas de conectividad con el servidor proporcionado por los docentes, el cual soporta tanto direccionamiento IPv4 como IPv6.

## Requisitos Previos

- Python 3 instalado en el sistema
- Acceso a terminal/consola
- Archivo del servidor (`server.py`) proporcionado por los docentes

## Instrucciones de Uso

### 1. Iniciar el Servidor

Para compilar y ejecutar el servidor, utiliza el siguiente comando en la terminal:

```bash
python3 server.py
```

### 2. Ir a ejecutar el cliente

Una vez lanzado el servidor, ir al programa de socket y ejecutarlo

### 3. Revisar consola

Cuando se lanza el programa del socket, en la consola del servidor aparece informacion del cliente.
