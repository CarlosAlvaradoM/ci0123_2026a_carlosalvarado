# Cliente SSL IPV6

Este readme explica los pasos necesarios para ejecutar el programa.

## Instrucciones para ejecutar el cliente SSl IPV6

Sigue estos pasos en orden para poner en funcionamiento el programa.

### 1. Configurar los parámetros

Estar conectado en la red del lab 3-5

### 2. Compilar con el Makefile

En una terminal, compilar:

```bash
make
```
Este comando compila el código y genera los ejecutables `./tcp-cli.out`

### 3. Ejecutar el programa

Una vez compilado, ejecuta el programa en la terminal:

```bash
./tcp-cli.out
```

### 5. Verificar la conexión

Revisa que la conexión se haya realizado correctamente.
